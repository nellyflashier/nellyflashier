## What I work on

I analyse sequencing data to answer biological questions. My experience covers:

- **Bulk RNA-seq** — differential expression analysis, pathway enrichment, and biological interpretation
- **Single-cell RNA-seq** — cell type annotation, trajectory analysis, and disease-relevant transcriptomic profiling
- **Spatial transcriptomics** — tissue-resolved gene expression analysis using 10x Visium and related platforms
- **Multi-omics integration** — combining transcriptomic and other omics layers to identify biologically meaningful patterns
